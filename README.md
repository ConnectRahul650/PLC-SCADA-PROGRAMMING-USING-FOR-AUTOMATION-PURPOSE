# PLC-SCADA-PROGRAMMING-USING-FOR-AUTOMATION-PURPOSE
PLC-SCADA based automated production line featuring filling, capping, labeling, inspection, batch counting, and HMI-based process monitoring.

## Overview

This project demonstrates an industrial automated filling system developed using **OpenPLC** for PLC programming and **ScadaBR** for SCADA/HMI monitoring. The system automates product loading, filling, capping, tightening, inspection, labeling, and batch counting operations. It supports multi-product selection through HMI and utilizes sensors, timers, counters, and safety interlocks to ensure efficient, reliable, and safe production line automation.

## Objectives

To automate the production line to manage different products

To Develop a PLC system that can switch between the produce the different products based on user input.

Improve process monitoring and control
